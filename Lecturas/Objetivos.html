INTRODUCCION

Este sitio web esta diseñado para explicar de forma basica y rapida todos los conceptos necesarios que se deben aprender para un manejo aceptable de HTML

OBJETIVO GENERAL

El objetivo de la presente guia de estudio es proporcionar las nociones
básicas y fundamentales acerca de la creación de paginas web con
el leguaje HTML, de esta manera, el usuario será capaz de crear su
propia página web con estrurturas básicas y sencillas que servirán
de soporte para su futuro aprendizaje.

OBJETIVO ESPECIFICO

  • conocer orden y el funcionamiento de las diversas etiquetas utilizadas en HTML
  • comprender la utilidad de lo atributos entro de las etiquetas
  • identificar los diferentes tipos de valores, su importancia, funcionamiento y relevancia ala hora de crear la estructura necesaria con HMTL
